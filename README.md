# 韩曼莉律师团队网站

这是韩曼莉律师团队的静态网站项目，重点展示韩曼莉律师个人专业形象、团队成员、法律产品服务、证书与聘书、A类特殊人群的人财保·传、私人定制服务等内容。

## 页面

- `index.html`：首页
- `han-manli.html`：韩曼莉律师详细资料
- `private-service.html`：高净值人群私人定制服务
- `product.html`：A类特殊人群的人财保·传产品
- `certificates.html`：证书与聘书完整展示
- `team.html`：团队律师详细介绍

## 资源

- `assets/`：网页图片、律师照片、证书缩略图
- `public-docs/`：网站公开访问所需的证书、聘书和产品手册文件

原始素材目录 `律师资料/` 未纳入 Git，用于避免上传申报表、简历原件等敏感源文件。

## 本地预览

```bash
python -m http.server 4173 --bind 127.0.0.1
```

然后访问：

```text
http://127.0.0.1:4173/index.html
```
